# 📜 Règles de la communauté MemeDrop

Ces règles sont la version par défaut affichée dans le client et via
`/guidelines` sur Discord. Elles sont **modifiables** depuis le panel
(**Guidelines**, réservé aux admins) et s'appliquent par instance.

En envoyant un meme, chaque membre s'engage à les respecter. Le non-respect
entraîne un avertissement, un retrait du droit d'envoi, puis un bannissement.

## ✅ Autorisé
- L'humour, les vannes entre potes, les memes, le second degré.
- **L'humour noir est toléré** — tant qu'il reste dans le respect des personnes.

## ⛔ Interdit
- Contenu **pornographique**, sexuellement explicite, ou tout contenu sexuel
  impliquant des mineurs (**tolérance zéro**, signalement immédiat).
- Contenu **raciste**, antisémite, homophobe, transphobe, validiste, ou toute
  **incitation à la haine** visant un groupe.
- **Harcèlement**, menaces, *doxxing*, contenu visant à nuire à une personne réelle.
- Violence gratuite, gore choquant, apologie du terrorisme.
- Contenu illégal, malveillant, ou destiné à tromper / arnaquer.

## 🤝 Bon sens
- Un meme doit faire rire, pas blesser. Dans le doute, s'abstenir.
- Respecte les cooldowns, ne spamme pas l'écran des autres.
- Les modérateurs peuvent retirer un contenu et sanctionner à tout moment.

## 🚨 Signaler
- **Client** : via la fenêtre Réglages / ou demande à un modérateur.
- **Discord** : commande `/report`.
- **Panel** : onglet **Modération** (modérateurs/admins).

---

## Note technique pour les administrateurs

Le filtre automatique fourni est un **point de départ** volontairement compact.
Pour une modération efficace :

1. **Étendez la liste** par channel : *Channel → Réglages → Mots interdits*
   (un mot/expression par ligne). La normalisation (accents, leetspeak,
   espacement, répétitions) est appliquée automatiquement, inutile de lister
   toutes les variantes.
2. Nommez des **modérateurs** (whitelist) : ils gèrent bannissements et retraits.
3. Activez le mode **« filtre + revue »** pour les communautés sensibles.
4. La modération automatique du **texte** est robuste ; la détection automatique
   d'images offensantes est *best-effort* — les modérateurs humains et le système
   de signalement restent la garantie principale.
