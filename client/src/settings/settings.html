👋 Welcome to MemeDrop

Three steps to receive your friends' memes on your screen:

  1. Get a pairing code — from the web panel (a channel's Devices tab), with the /link Discord command, or ask a moderator.
  2. Paste the server URL + the code below, then click Pair.
  3. Test the placement of the overlay (Overlay tab) and you're good to go 🚀
